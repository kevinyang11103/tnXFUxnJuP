# 智能停车计费管理系统 java708

## 项目概述

智能停车计费管理系统是一款基于SpringBoot+Vue前后端分离技术构建的停车管理系统。系统致力于提供高效、便捷的车位管理，车辆进出控制，以及完善的计费和惩罚机制。

## 技术栈

- 后端：SpringBoot
- 前端：Vue
- 数据库：MySQL
- 开发工具：Idea，vscode
- 其他：mybatis-plus，element UI

## 功能模块

### 系统角色

- 用户
- 超级管理员

### 功能详情

- **车位信息管理**：管理车位的分类、状态等信息。
- **车子进场管理**：记录车辆进场时间，分配车位。
- **车子离场管理**：处理车辆离场流程，计费管理。
- **违规惩罚管理**：对违规停车行为进行记录和惩罚。
- **公告管理**：发布和更新停车场的公告信息。
- **论坛管理**：提供用户交流的平台，管理论坛内容。
- **车位分类管理**：设置和调整车位分类。
- **个人中心**：用户可查看个人信息和停车记录。
- **个人后台**：超级管理员管理后台，进行系统维护。

## 运行环境

- 后端：Java 1.8+环境
- 前端：支持现代Web浏览器
- 数据库：MySQL 5.7+
